BGL API

BGL API

The BGL API is a set of RESTful Web Services using OAuth 2.0 protocol as the security mechanism. The API provides access to BGL users' Data to client applications with the users' consent.

The objective of this document is to provide information on the BGL API set-up and the resources required to request information through the BGL API to developers.
